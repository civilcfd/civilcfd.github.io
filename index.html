About

Civil-CFD is a free surface 3D CFD Solver for Civil Engineering Problems, including:
  • Meshing tool that takes STL files as input to define irregular geometries
  • Solver with VOF algorithm for free surface flow, and k-E turbulence model
  • Parallel solver using PetSc and MPI
  • Graphical interface for rapid case development and visualization
  • Integration with Paraview for more complex visualizations
